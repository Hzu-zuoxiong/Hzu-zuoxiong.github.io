Git 常见文件

gitkeep 文件

Git 是不允许提交一个空的目录到版本库上的,可以在空的文件夹里面建立一个 .gitkeep 文件，然后提交去即可。其实在 Git 中 .gitkeep 就是一个占位符。

gitignore 文件

当我们需要将一个项目提交到 Git 时，并不是所有的文件都需要提交，比如一些自动生成的文件，这时候就可以使用 .gitignore 来忽略一些不需要提交的文件。
